# Docs
- npm config set registry https://npm.pkg.github.com/patternsandbox
- echo "//npm.pkg.github.com/:_authToken=TOKEN" > ~/.npmrc
- add `"@patternsandbox/docs": "1.0.12",` to your package.json
- npm install

Read more: [Github Packages]
